Maguaré

A colombian government website targeted at kids in the early childhood. This site was very slow and had a legacy Javascript code with really bad practices.

My job was to rewrite the Javascript engine making it backward compatible so that all contents, activities, videos and audios would still work. After I rewrote the engine, the site loaded over 66% faster and made less than 3% of request to the server.
